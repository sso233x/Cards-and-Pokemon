## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/Cards-and-Pokemon.git
   cd Cards-and-Pokemon
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
